# חיבור המחשב שלך למאגר הידע של אוטוטיוזדיי

> 💡 **הכי פשוט:** במקום הצעדים הידניים למטה — ראה `CONNECT-PROMPT.md`: פרומפט אחד
> שמדביקים ל-Claude Code והוא מתקין ומחבר את עצמו (Windows/Mac). הידני למטה = לגיבוי.


> 5 דקות. אחרי זה ה-Claude Code שלך "מדבר" עם הידע הארגוני — החלטות, סטטוס, פרויקטים —
> ורואה **בדיוק** את מה שמותר לך (המערכת מזהה אותך אוטומטית).

## מה צריך לפני
- **Node 18+** (בדיקה: `node -v`)
- **Claude Code** מותקן
- חשבון בפורטל `app.autotuesday.com` (אתה כבר רשום)

## צעד 1 — התקנה (פעם אחת)
```bash
git clone https://github.com/netanelhibsh/autotuesday-kb-mcp.git
cd autotuesday-kb-mcp
npm install
npm run build
```
שים לב לנתיב המלא של `dist/index.js` (תצטרך אותו בצעד 3). למשל:
`/Users/<שם>/autotuesday-kb-mcp/dist/index.js`

## צעד 2 — הגדרת סיסמת מחשב + קבלת ה-config
1. היכנס בדפדפן ל-**`app.autotuesday.com/me/desktop`**
2. ב"צעד 2 — הגדרת סיסמת מחשב": לחץ **"🎲 צור לי סיסמה"** (או בחר בעצמך) → **"הגדר סיסמת מחשב"**.
   (זו סיסמה ייעודית לחיבור המחשב — לא משנה את הכניסה הרגילה בגוגל/מייל.)
3. ייפתח בלוק `config` מוכן עם המייל והסיסמה שלך. **העתק אותו** (כפתור "📋 העתק את ה-config").

## צעד 3 — הדבקה ב-Claude Code
פתח את `~/.claude/settings.json` והוסף/מזג את הבלוק `mcpServers` שהעתקת.
**עדכן את הנתיב** ב-`args` ל-`dist/index.js` שלך מצעד 1.
> הסיסמה עובדת מכל מחשב — מעבר למחשב אחר = אותו config, אותה סיסמה.

## צעד 4 — הפעלה מחדש ובדיקה
1. סגור ופתח מחדש את Claude Code.
2. שאל:
   > "מה ההחלטות האחרונות של אוטוטיוזדיי?"
   או:
   > "השתמש ב-whoami של autotuesday-kb — מי אני?"
3. אמור לענות מהמאגר, ולזהות אותך בשמך ובתפקידך. ✅

## מה יש לך עכשיו (הכלים)
| כלי | מה |
|-----|-----|
| `kb_search` | חיפוש טקסט בידע |
| `kb_list` | רשימת פריטים (לפי פרויקט/סוג) |
| `kb_read` | קריאת פריט מלא |
| `kb_write` | כתיבה/עדכון (מה שמותר לך) |
| `kb_history` / `kb_rollback` | גרסאות + החזרה אחורה |
| `workspaces_list` | הפרויקטים שיש לך גישה אליהם |

## תקוע?
- "no auth" / "password auth failed" → ודא שהמייל+סיסמה ב-config נכונים (הגדרת סיסמה ב-/me/desktop?) והנתיב ל-dist/index.js נכון.
- לא רואה פריט? → כנראה הוא לא בהרשאה שלך (פרטי/פרויקט אחר). זה תקין.
